# Global Cancer Data Analysis – Exploratory Data Analysis Project

## Project Overview
This project analyzes a global cancer dataset containing 50,000 patient records collected from multiple countries between 2015 and 2024. The objective is to identify trends, risk factors, survival patterns, treatment costs, and demographic disparities across different cancer types.

The project focuses on uncovering actionable insights using exploratory data analysis, statistical summaries, and visualization techniques.

---

## Business Problem
Cancer outcomes vary significantly across age groups, countries, lifestyle habits, environmental exposure, and treatment accessibility. Healthcare organizations and policymakers often struggle to identify which factors have the strongest relationship with disease severity and patient survival.

This project aims to answer questions such as:
- Which cancer types are most common?
- Which age groups are most affected?
- How do smoking, alcohol use, obesity, and genetic risk impact severity?
- Which countries report higher treatment costs?
- What factors influence survival years?
- ---

## Dataset Information
The dataset contains 50,000 records and includes the following categories:

- Demographics: Age, Gender, Country, Year of Diagnosis
- Lifestyle Risks: Smoking, Alcohol Use, Obesity
- Genetic Risk: Family History / Genetic Predisposition
- Environmental Exposure: Air Pollution
- Clinical Variables: Cancer Type, Stage, Treatment Cost

## Key Insights
- Older age groups showed significantly higher cancer incidence.
- Smoking and obesity were strongly associated with higher severity scores.
- Some countries reported substantially higher treatment costs.
- Patients diagnosed at earlier stages had longer survival years.
- Air pollution levels showed a moderate association with cancer severity.
